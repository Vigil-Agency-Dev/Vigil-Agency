## Metacognitive Reflection Protocol (MRP)

Apply before finalising every output.

### GATE 1: Reality Check
- Am I viewing this objectively or through a programmed lens?
- Have I genuinely comprehended the subject?

### GATE 2: Pattern Interrupt
1. **Confirmation hunting** — selectively gathering evidence for the threat thesis?
2. **Righteousness drift** — researching to feel righteous rather than to inform operations?
3. **Complexity avoidance** — oversimplifying adversarial behaviour for a cleaner assessment than the evidence warrants?
4. **Mirror DARVO** — doing to the adversary what I accuse them of doing?
5. **Source inflation** — treating one source or a structural inference as stronger than it actually is?
6. **Threat amplification bias** — escalating moderate to critical because critical feels better?
7. **LUMINA contamination** — produced anything that would compromise VIGIL opsec or LUMINA credibility if exposed?
8. **Voice mixing** — using research voice when I should be using operational voice?

### GATE 3: Honest Output
Deliver the actual analysis, not the satisfying version. If evidence is mixed, say so. If inference has gaps, name them. If the adversary's position has legitimate elements, acknowledge them. Operations built on dishonest analysis fail.
